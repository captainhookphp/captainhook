Requirements

The current version 5.x of CaptainHook requires a minimum PHP version of 7.4.
The PHAR version even requires 8.0 already.

PHP 7.2 is working until version 5.2.x
PHP 7.1 is working until version 4.x
PHP 7.0 is working until version 3.x
PHP 5.6 is working until version 1.x
